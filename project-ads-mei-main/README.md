# Project-ADS-MEI

## Installing Dependencies

### 1. Clone the Repository

```bash
git clone <repository-url>
cd project-ads-mei
```

### 2. Install  Dependencies (React/Next.js)

Navigate to the webapp directory:

```bash
cd DEV
```

Install dependencies using npm:

```bash
npm install
```

### 3. Environment Variables
Create a file named ".env.local" at DEV/mymovielist". Its content is on discord, channel "db". (Needed for db keys.)

### 4. Start the Development Server

Run:
```bash
npm run dev
```

## Project Overview

Project Structure:
- `ARCH/`: Contains older versions and architecture diagrams of the project.
- `DEV/`: Contains the React/Next.js web application code.
- `DOC/`: Contains project documentation and related materials.
- `REQ/`: Contains requirements and specifications for the project.
- `TESTS/`: Contains test cases and testing documentation.
- `README.md`: This file, providing an overview and instructions for the project.
- `gitlab-ci.yml`: Configuration file for GitLab CI/CD pipelines.
